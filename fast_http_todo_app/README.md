# Golang To Do App

This project is to do app using fiber and sqlx

## Requirements

 - Go >= 1.15.x
 - PostgreSQL
 - Docker

## Getting Started

 ```
 git clone https://github.com/ayberkbozkus/Golang/tree/main/fast_http_todo_app.git
 docker build --tag fast-http-todo .
 docker run -d -p 3000:3000 fast-http-todo
 ```
